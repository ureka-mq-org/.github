# 🚀 Ureka MQ Study Group

**ureka-mq-org**는 다양한 메시지 큐 솔루션을 비교 분석하고, Spring 생태계 기반에서 RabbitMQ와 Kafka의 실전 구현 및 최적화 방안을 탐구하는 스터디 조직입니다.

## 📌 Study Goals
* **MQ Analysis**: PGMQ, Redis MQ, RabbitMQ, Kafka 등 다양한 MQ 솔루션의 핵심 개념 및 특징 파악
* **Core Implementation**: Spring 환경에서 RabbitMQ와 Kafka를 활용한 메시징 인프라 구축 및 실전 구현
* **Performance Optimization**: 메시지 처리량 최적화 및 안정성 확보를 위한 성능 튜닝 연구
