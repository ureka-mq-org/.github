## 📌 Project Goals
* 🔍 **MQ Analysis**
  * PGMQ, Redis MQ, RabbitMQ, Kafka 등 솔루션별 핵심 개념 및 특징 파악
* 🛠️ **Core Implementation**
  * Spring 생태계 내 RabbitMQ와 Kafka를 활용한 메시징 인프라 구축 및 구현
* ⚡ **Performance Optimization**
  * 메시지 처리량 최적화 및 안정성 확보를 위한 성능 튜닝 연구

<br/>

## 📂 Repositories
> **sapari 프로젝트의 이론 정리와 실습 코드는 아래 레포지토리에서 관리됩니다.**

* 📖 **[core-concept](https://github.com/sago-panda/core-concept)**
  * MQ 관련 이론 정리 (솔루션 비교 분석 및 브로커별 핵심 동작 원리 학습)
* 🛠️ **[mq-study](https://github.com/sago-panda/mq-study)**
  * Spring Boot 기반 실무 예제 및 최적화 설정(Prefetch, Batch 등) 성능 검증
